# Política de Seguridad

## Versiones soportadas

Actualmente, este proyecto se mantiene únicamente en su versión principal publicada en este repositorio.

| Versión | Soporte |
|--------|---------|
| Rama principal / main | Soportada |
| Versiones anteriores | No soportadas |

## Reporte de vulnerabilidades

Si identificas algún problema de seguridad relacionado con este proyecto, puedes reportarlo mediante un issue en el repositorio o contactando directamente a la autora.

Por favor, incluye en el reporte:

- Descripción clara del problema.
- Pasos para reproducirlo, si aplica.
- Archivo o sección del proyecto donde se presenta el problema.
- Evidencia o captura, si es necesario.

## Alcance

Esta política aplica únicamente al código fuente, notebook, documentación y archivos incluidos en este repositorio.

El conjunto de datos utilizado para el entrenamiento no forma parte del repositorio, por lo que no está incluido dentro del alcance de esta política.
